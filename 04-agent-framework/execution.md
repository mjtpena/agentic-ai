# 3. Emerging Standards (Critical Development)

## 3.1 MCP (Model Context Protocol)

**Status (2026): Early-stage but rapidly gaining adoption**

MCP allows:
- Standardized context exchange
- Tool descriptions
- Memory injection
- Agent-to-tool interoperability

Backed by:
- OpenAI
- Anthropic
- Multiple open-source groups

Impact:
Reduces vendor lock-in and tool integration complexity.

---

## 3.2 A2A (Agent-to-Agent Protocol)

Emerging standard for:
- Multi-agent communication
- Role-based coordination
- Task delegation
- Memory passing

Strong research backing from:
- Microsoft AutoGen team
- Google DeepMind
- Open-source communities

Still fragmented — no single dominant implementation yet.

## 3.3 Function Calling as De Facto Standard
